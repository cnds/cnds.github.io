Python的传值

Python函数的传值分两种，对于不可变对象，是值传递(call by value)，对于可变对象，是引用传递(call by reference)

上面的代码可以看到，对于不可变对象，在函数中改变了num的值并不会影响d, 同时num的id也发生了变化，这说明传到函数中的d的一个拷贝。

对于可变对象，在函数中改变num，外面的d也发生了改变，num的id并没有发生变化,这说明传到函数中的值是d的引用。
